Observações Importantes
--------------------------

* **Validação do DV:** O algoritmo @dv_mod11_executivo_federal_2d@ é o mesmo utilizado pelo Governo Federal, garantindo plena compatibilidade com o SEI.

* **Identificador de Origem (17):** No caso do Tocantins, este número é fixo para todo o estado do Tocantins. Municípios e unidades que pertencem ao governo Estadual do Tocantins devem sempre iniciar com 17, alterando apenas o código da instituição vinculada (os 3 dígitos seguintes).
